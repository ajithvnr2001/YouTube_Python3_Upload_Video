## What's New in the Script

### Multi-Caption Support
The script now accepts **unlimited caption files** in a single upload command:[3][1]

```python
--captions lang:filepath lang:filepath lang:filepath ...
```

### Two Caption Formats Supported

**Format 1: Simple** (language:filepath)
```bash
--captions en:english.srt es:spanish.srt hi:hindi.srt
```

**Format 2: With Custom Names** (language:name:filepath)
```bash
--captions en:"English CC":english.srt es:"Español":spanish.srt
```

## Quick Start Examples

### Example 1: Basic Multi-Language Upload
```bash
python upload_with_captions.py \
    --file="movie.mp4" \
    --title="My Movie" \
    --description="Available in 5 languages" \
    --captions \
        en:subtitles/english.srt \
        es:subtitles/spanish.srt \
        fr:subtitles/french.srt \
```

## Installation & Setup

### 1. Install Dependencies
```bash
pip install --upgrade google-api-python-client oauth2client httplib2
```

### 2. Get OAuth Credentials
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a project and enable YouTube Data API v3
3. Create OAuth 2.0 Client ID credentials
4. Download `client_secrets.json` to script directory

### 3. Run the Script
```bash
python upload_with_captions.py --file="video.mp4" --captions en:english.srt
```

**Multiple Captions in One Command**:[1][3]
